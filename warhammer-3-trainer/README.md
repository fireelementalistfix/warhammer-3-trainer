# Warhammer 3 Trainer

A simple console-based trainer for Total War: Warhammer III.

Features:
- Set player gold
- Set player influence
- Unlimited movement (toggle)
- God mode (toggle)

## Usage

Run the trainer while the game is running. Use the menu options to activate cheats.

## Build

Requires .NET 6.0 or higher.

```
dotnet build
dotnet run --project warhammer-3-trainer
```

## Tests

```
dotnet test
```
